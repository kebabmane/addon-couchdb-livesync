# CouchDB for Obsidian LiveSync

CouchDB database pre-configured for the Obsidian LiveSync plugin, enabling self-hosted synchronization of your Obsidian notes across all your devices.

## About

This add-on runs CouchDB with all the necessary configuration for [Obsidian LiveSync](https://github.com/vrtmrz/obsidian-livesync), including:

- CORS settings for Obsidian app
- Increased document size limits for large notes
- Single-node configuration optimized for personal use
- Persistent data storage

## Quick Start

1. Install the add-on
2. Configure your username and password
3. Start the add-on
4. Connect Obsidian LiveSync plugin to `http://<your-home-assistant-ip>:5984`

See the [Documentation](DOCS.md) tab for detailed setup instructions.
