# CouchDB LiveSync Add-on Repository

Home Assistant add-on repository for CouchDB configured for Obsidian LiveSync.

## Add-ons

### [CouchDB for Obsidian LiveSync](./couchdb-livesync)

CouchDB database pre-configured for the Obsidian LiveSync plugin, enabling self-hosted synchronization of your Obsidian notes.

## Installation

1. Navigate to **Settings** > **Add-ons** in Home Assistant
2. Click the three dots in the top right corner
3. Select **Repositories**
4. Add this repository URL: `https://github.com/kebabmane/addon-couchdb-livesync`
5. Click **Add**
6. Find "CouchDB for Obsidian LiveSync" in the add-on store
7. Click **Install**
